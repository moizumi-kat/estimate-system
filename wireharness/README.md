# ワイヤーハーネス製作支援システム v0.1

制御盤・分電盤の**結線図**をアップロードすると、Claude Vision が「電線1本ごとの接続（From-To）」を抽出し、
ワイヤーハーネス製作に必要な次のデータを自動生成します。

| 出力 | 内容 |
|------|------|
| **電線リスト** | 号線・接続元/先（機器記号:端子）・電線種類/サイズ/色・長さ・備考 |
| **切断・端末加工表** | 電線ごとの切断長・両端の端子（圧着）・マーク（号線） |
| **結束・製作図** | 幹線に沿った機器分岐と区間ごとの結束本数を図化した SVG |
| **部材集計** | 電線（種類/サイズ/色 別 総長）・端子（種類別 本数） |

積算コード選定システム（リポジトリ直下 `app.py`）とは**独立した別アプリ**です。

## 使い方

```bash
cd wireharness
pip install -r requirements.txt
export ANTHROPIC_API_KEY=sk-ant-...
python3 app.py            # → http://localhost:8010
```

1. 結線図（端子台結線図・展開接続図・実体配線図。PDF / PNG / JPG / DXF / ZIP）をアップロード
2. 「① 結線を抽出」で電線の From-To を読み取り
3. 電線リストを確認・編集（機器・端子・電線種・サイズ・色・長さ）
4. 「切断長・端子・製作図を再計算」で切断長／端子候補／製作図／集計を更新
5. 「Excelで保存」で 3 シート（電線リスト / 切断・端末加工表 / 部材集計）を出力

## 設計方針

- AI は**図面に描かれている結線だけ**を読み取り、存在しない配線を創作しません。
- 読み取りが曖昧な行は「要確認」（unclear）として色分け表示します。
- 端子（圧着）はサイズからの推定候補です（例: 1.25sq → R1.25-4）。画面で修正できます。

## 環境変数（任意）

| 変数 | 既定 | 説明 |
|------|------|------|
| `ANTHROPIC_API_KEY` | （必須） | Claude API キー |
| `PORT` | `8010` | 待受ポート |
| `SELECT_MODEL` | `claude-opus-4-8` | 抽出に使うモデル |
| `APP_PASSWORD` | （空=認証オフ） | 設定するとパスワード認証を有効化 |
| `WH_END_ALLOW` | `50` | 端末代（片端 mm）。切断長 = 電線長 + 2×この値 |
| `WH_STUD` | `4` | 端子の既定ねじ径（M4 → `R1.25-4`） |
