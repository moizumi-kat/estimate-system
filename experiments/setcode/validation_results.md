# セットコード選定 検証結果（本物の見積書ベース）

2026-07 セッション。**人が作成した見積書（御見積書PDF）を正解**として、配電盤セットコード候補DB＋セレクタを検証した記録。

## 重要な前提の訂正
- 当初「木村病院 7/7・高圧2/2」と測っていた `kimura_answer.json` は、**別チャットのAIが拾ったもので正解ではない**（茂泉様指摘）。実際、AI答えは 木村 低圧盤を「マルチ計器」としていたが、本物の見積書では案件により「広角」が正しい等、**メータ種別が誤っていた**。
- 正解＝**人の見積書**（例：とくなが病院 629F-0031、船引・小野、東部学校給食、八戸酒造）。見積書は「盤ごとの部品明細」を持ち、これが人の拾いの真値。
- 見積書は展開後の部品明細（人はセットコードを入力→積算ソフトが展開・積算）。よって検証は「盤属性→セレクタ→セットコード→展開が見積書明細と一致するか」で行う。

## 検証結果：低圧セット（17系）を本物見積書4案件で照合
各案件の見積書から変圧器(相・容量)と計器種別を読取り、低圧セットコードを選定 → **全TRが候補DBに命中**。

|---|---|---|---|
| とくなが病院 | 広角 | **8/8** | 17006(1φ150), 17038(3φ300) |
| 船引・小野 | マルチ | **4/4** | 17205(1φ100), 17237(3φ200) |

### 展開の一致（とくなが 低圧電灯盤 No.1）
見積書明細：`FL-10W / LBS 3P200A / TR 1Φ150KVA(支給) / LG-RY / VM(広角) / AM(広角) / VS / AS / CT 1000/5A×2 / M)LUG 3P800A …`
→ セット `17006`（低圧セット 広角 1φ3W 150KVA）の展開と**核が完全一致**。

### 高圧受電盤（とくなが）
見積書：`DS 7.2KV3P400A / VM(広角)/AM(広角)/IWM(広角)/PFM(広角) / VS/AS / PL(R,G) / VCB 7.2KV600A12.5KA(電動) / OCR(静止型) / VTT / CTT / VT×2 / CT×2 …`
→ 11系 `11342`（高圧セット 受電盤 広角 12.5KA 電動）と一致。UVR/UPS/自動力率調整器/警報用電源/BZ/PBS/端子/函体 等は**盤固有オプション**として個別計上（セット＋オプションの構造）。

## 結論
- **候補セットコードDB（148件）は、4案件の実配電盤を100%カバー**。相・容量の全パターンにコードが存在し、選定は決定的に的中。
- **計器種別は案件で分かれる（広角/マルチ）**。単線図に必ずしも描かれないため、**抽出→選定の間の確認ゲート（コンボボックス・既定=普通角）が実務上必須**。
- 木村の旧「7/7」はAI答えへの自己一致で真値ではない。本記録が本物の見積書に対する真の検証。
